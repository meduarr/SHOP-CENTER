Nos últimos dias me aprofundei em banco de dados e tive a oportunidade de desenvolver um projeto web integrado ao MySQL, utilizando PHP, HTML, CSS e JavaScript.

Foi uma experiência muito enriquecedora, que me permitiu entender melhor a comunicação entre a aplicação e o banco de dados, além de fortalecer meus conhecimentos em desenvolvimento web.

O projeto conta com funcionalidades como:
✅ Cadastro de usuários
✅ Sistema de login
✅ Recuperação de senha
✅ Integração com banco de dados MySQL
✅ Hospedagem online

Além do desenvolvimento, também publiquei o projeto no GitHub e realizei sua hospedagem na web, fortalecendo meus conhecimentos em versionamento, deploy e gerenciamento de banco de dados online.
